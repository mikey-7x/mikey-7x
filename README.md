### 👨‍💻 About Me

I am a practical electronics engineer specializing in **analog design, high-power circuitry, and Radio Frequency (RF) systems**. My approach to circuit design relies heavily on hands-on experience and intuitive diagram memorization, allowing me to calculate approximate values through direct experimentation rather than relying solely on theoretical mathematical analysis.

- ⚙️ **Embedded Systems:** I work deeply in register-level programming, bootloader configurations, and assembly language math across **ATmega, STM32, and ESP8266** platforms.
- 💻 **Architecture & Emulation:** I am passionate about emulating retro 8-bit CPUs on modern hardware to gain a profound, ground-up understanding of core processing architectures.
- 🐧 **Mobile & Edge Linux:** I leverage **Termux** on high-end Android hardware (Snapdragon 8 Gen 3) to host full Linux environments (Ubuntu & Arch via Proot), complete with GPU hardware acceleration and X11 display servers for on-the-go machine learning scripting. (I also run WSL 1 environments for lighter desktop development).
- ⚖️ **Open Source Philosophy:** I build tools for the community. My public projects utilize custom non-commercial licenses that allow educational use while strictly requiring name attribution for modifications and prohibiting for-profit sales.

### 🚀 Featured Projects

- 🧠 **Project Synapse:** A hardware-software integration for advanced biosignal processing. It features a custom hardware front-end designed for electromyography (EMG) and subvocal recognition, paired with machine learning training for precise signal decoding.
- 🏭 **Industrial Faultshield:** A comprehensive telemetry and monitoring system for industrial motors. It integrates multiple physical sensors and transmits diagnostic data via Bluetooth to a custom-built Python GUI.
- 🚗 **Driver Safety System (DSS4):** An Advanced Driver Assistance System (ADAS) prototype utilizing computer vision alongside custom PWM steering logic and motor control scripts running on ATmega microcontrollers.
